Big data set

When dealing with big data files, do not read everything into memory. We use the sample data from: http://grouplens.org/datasets/hetrec-2011/ See: http://files.grouplens.org/datasets/hetrec2011/hetrec2011-movielens-readme.txt

## give the input filename
data.path = 'hetrec2011'
ifilename = sprintf('%s/movies.dat',data.path)
## read the first n=3 lines
lines = readLines(ifilename,n=3)
